## vscode 多行快捷键

| 快捷键 | 作用 |
|-- |--|
| ctrl+alt+上下键 | 进行 行选择  |
| shift+左右键 | 进行 列选择  |
| 推荐: Ctrl + Alt + up/down | 上下插入光标 - 支持多个 |
|  |  |
| 选中一段文字，按shift+alt+i | 在每行末尾都会出现光标 |
|  |  |
| ctrl+左右键 | 自动识别一个整体 |
| ctrl+shift+左右键 | 自动识别并选择 |
| ctrl+shift+L | 选中编辑代码中相同的内容 |
|  |  |
| alt+鼠标点选 | 多处选择编辑 |
|  |  |
| ctrl+alt+左右键 | 分页 |
|  |  |
| 选中文本后,Ctrl + [ 和 Ctrl + ] | 可实现文本的向左移动 和 向右移动 |
